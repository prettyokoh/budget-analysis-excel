# Key Insights
Highest expenses were identified in specific categories
Some categories exceeded the planned budget
Clear spending patterns were observed

## Tools Used
Microsoft Excel
Pivot Tables
Charts

## Objectives
This analysis evaluates budgeted versus actual spending to identify cost control gaps, spending efficiency,and maajor cost drivers across departments nd categories.


## Key Insights
Engineering and Sales show the highest overspending.
Only one department achieved a positive variance, indicating strong cost control.
Spending trends are stable across months with no major spikes.
Higher headcount departments generally have lower spend per employee.
One major category dominates total expenditure.
